About Me

A Graphic Designer and Front End Developer!

I'm the complete package, a creative developer AND a designer!

Hi! I'm Mishara! A designer and developer. Since I was young, I've always had a passion to create things. After graduating from The University of Michigan-Ann Arbor with my B.F.A in Graphic Design, I started my journey towards being an awesome designer. As a Graphic Designer, I've created print materials for many companies and non-profits for over 10 years. I like to think that I can bring my customer's visions to life.

In the pursuit of being an AWESOME designer, I'm always learning new things to improve my skills, so I decided to start building website. I started off with simple Wordpress sites, only changing the CSS code to customize it but I wanted to know more. Basic knowledge of HTML and CSS would not get me far and I wanted to take this web design thing to another level. In March 2017, I decied to participate in a Front-end Developer Bootcamp at Grand Circus-Detroit. Grand Circus Front End Development Bootcamp, is an intense 10 week course that teaches you JavaScript, jQuery, AngularJS, Node and a host of other frameworks and libraries.

I'm currently still working as a Graphic Designer and building websites/ applications, while learning new languages such as SASS and React.
